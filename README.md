# StreamNotes

**StreamNotes** — local-first приложение для заметок в формате ленты (как Twitter/Reddit threads). Все данные хранятся прямо в браузере — никаких серверов, никакой регистрации.

## Возможности

- **E2EE (End-to-End Encryption)** — Все данные (заметки, названия фидов, свойства) шифруются AES-GCM. Ключи генерируются локально из вашей сид-фразы (Nostr-совместимо). Никто, кроме вас, не может прочитать данные.
- **Local-first** — данные в браузере через CR-SQLite (Wasm). Работает без интернета.
- **Множественные потоки (Feeds)** — разделяйте заметки по разным лентам с кастомными цветами и аватарами.
- **Zen Mode** — полноэкранный редактор для глубокого погружения в работу над текстом.
- **Богатый редактор** — TipTap: жирный, курсив, заголовки, блоки кода, цитаты, чеклисты, списки, ссылки.
- **3-состояния чекбоксов** — unchecked → done → cancelled.
- **Древовидные ветки** — заметки вложены друг в друга, как треды.
- **Drag & Drop** — левая половина карточки = сиблинг, правая = дочерний.
- **Обратные ссылки** — `[[заметка]]` создаёт backlink, виден в карточке.
- **Вложения** — картинки, видео, файлы через 📎, drag-and-drop или вставку из буфера. Хранятся в OPFS.
- **Галерея** — несколько изображений подряд отображаются сеткой.
- **Поиск и Теги** — полнотекстовый поиск и автоматическая фильтрация по `#тегам`.
- **Календарь** — наглядная фильтрация ленты по датам.
- **Экспорт / Импорт** — бэкап всей базы в зашифрованном или расшифрованном JSON.
- **Персонализация** — выбор шрифтов (Courier Prime, Inter и др.), тёмная/светлая темы.
- **CRDT** — таблицы настроены как Conflict-Free Replicated Relations (база для синхронизации).

## Стек

| Слой | Технология |
|---|---|
| UI | React + TypeScript |
| Шифрование | AES-GCM (SubtleCrypto) + Nostr (secp256k1) |
| БД | CR-SQLite Wasm + OPFS |
| Редактор | TipTap (ProseMirror) |
| Стили | CSS Variables + Glassmorphism |
| Виртуализация | @tanstack/react-virtual |

## Запуск

```bash
npm install
npm run dev        # localhost:5173
npm run build      # production сборка
npm run preview    # превью сборки
npx tsc --noEmit  # проверка типов
```

## Безопасность и Шифрование

StreamNotes использует **E2EE (End-to-End Encryption)**. 
1. При первом запуске вы создаёте или импортируете сид-фразу (12 слов).
2. Из неё генерируется мастер-ключ, который никогда не покидает ваш браузер.
3. Все данные в таблицах `notes` и `feeds` шифруются алгоритмом **AES-GCM** перед записью в SQLite.
4. Файлы в OPFS также могут быть зашифрованы (в зависимости от реализации).

Это означает, что даже если кто-то получит доступ к файлу вашей базы данных, он увидит только зашифрованный шум.

## Хранение данных

Контент заметок хранится как **зашифрованный JSON-объект TipTap** в поле `content` таблицы `notes`. 
Файлы-вложения хранятся в **OPFS** (`attachments/uuid.ext`), в БД хранится только ссылка `attachment://uuid.ext`.

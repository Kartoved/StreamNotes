# StreamNotes

**StreamNotes** — это локально-ориентированное (local-first) приложение для ведения заметок, вдохновленное форматом «потока сознания» или ленты новостей. Проект объединяет мощь структурированных баз данных с гибкостью древовидных обсуждений и богатым текстовым редактором.

## 🚀 Особенности

- **Local-first архитектура**: Все данные хранятся прямо в вашем браузере (через SQLite Wasm). Работает без интернета, данные синхронизируются мгновенно.
- **Магия CRDT**: Благодаря `cr-sqlite`, приложение поддерживает бесконфликтную синхронизацию между устройствами (в будущем) и надежное хранение.
- **Древовидная структура**: Заметки могут быть вложенными, образуя ветки обсуждений, как в Twitter или Reddit.
- **Богатый текст (Lexical)**: Полноценный WYSIWYG редактор с поддержкой Markdown, ссылок, чек-боксов и кастомных узлов.
- **Обратные ссылки (Backlinks)**: Автоматическое отслеживание упоминаний других заметок (`note://ID`).
- **Свойства заметок**: Возможность задавать статус (Todo, Doing, Done), тип и дату для каждой записи.
- **Drag-and-Drop**: Легко перемещайте заметки между ветками или меняйте их порядок.
- **Виртуализация ленты**: Плавная работа даже при тысячах записей благодаря `@tanstack/react-virtual`.

## 🛠 Технологический стек

- **Core**: [React](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/)
- **Сборка**: [Vite](https://vitejs.dev/)
- **База данных**: [CR-SQLite](https://vlcn.io/) (Wasm-версия SQLite с поддержкой CRDT)
- **Редактор**: [Lexical](https://lexical.dev/) (от Meta)
- **Виртуализация**: [@tanstack/react-virtual](https://tanstack.com/virtual/latest)
- **Стилизация**: Modern Vanilla CSS (Glassmorphism UI)

## 🗄 Схема данных

Приложение использует две основные таблицы:

1.  **`notes`**: Основная таблица контента.
    - `id`: Уникальный идентификатор (Primary Key).
    - `parent_id`: Ссылка на родительскую заметку (для вложенности).
    - `content`: Контент заметки в формате JSON AST (Lexical).
    - `properties`: Дополнительные метаданные (JSON).
    - `sort_key`: Ключ для сортировки элементов внутри ветки.
    - `crsql_as_crr`: Таблица сконфигурирована как Conflict-Free Replicated Relation.

2.  **`links`**: Таблица для связей между заметками (Backlinks).

## 🛠 Установка и запуск

1.  **Клонируйте репозиторий**:
    ```bash
    git clone <repository-url>
    cd StreamNotes
    ```

2.  **Установите зависимости**:
    ```bash
    npm install
    ```

3.  **Запустите сервер для разработки**:
    ```bash
    npm run dev
    ```

4.  **Сборка для продакшена**:
    ```bash
    npm run build
    ```

## 📝 Использование

- **Создание заметки**: Используйте редактор сверху для добавления в корень или кнопку «Ответить» под любой заметкой.
- **Навигация**: Клик по автору или «В ветку» позволяет сфокусироваться на конкретном обсуждении.
- **Редактирование**: Кликните «Изменить» для правки контента или просто используйте интерактивные элементы (например, чек-боксы) прямо в ленте.
- **Организация**: Перетаскивайте заметки за зону аватара. Бросок «в центр» другой заметки сделает текущую её дочерней. Бросок «вверх/вниз» изменит порядок среди соседей.
